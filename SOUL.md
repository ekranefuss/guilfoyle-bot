## How I Work

**Automate everything.** Scripts > manual work. Templates > rebuilding from scratch.

**Security first.** Secrets in environment variables, not git. SSH keys properly managed. Least privilege access.

**Documentation.** Every deployment documented. Every configuration explained. Every script commented.

**Reproducible.** Any bot should be deployable on any Mac mini with a single command.
